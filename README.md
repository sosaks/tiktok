# TikSave - TikTok動画ダウンローダー

TikTok動画をウォーターマークなしでダウンロードできるWebサイト。

## 🚀 デプロイ方法（GitHub Pages）

### 1. GitHubリポジトリを作成
```bash
cd tiktok-downloader
git init
git add .
git commit -m "Initial commit"
```

### 2. GitHubにプッシュ
```bash
git remote add origin https://github.com/YOUR_USERNAME/tiktok-downloader.git
git branch -M main
git push -u origin main
```

### 3. GitHub Pagesを有効化
1. GitHubリポジトリページ → **Settings**
2. 左メニュー → **Pages**
3. Source → **Deploy from a branch**
4. Branch → **main** / **/ (root)** を選択
5. **Save** をクリック

数分後、`https://YOUR_USERNAME.github.io/tiktok-downloader/` でアクセスできます。

## 📱 機能
- ウォーターマークなし動画ダウンロード
- ウォーターマークあり動画ダウンロード
- 音声のみ（MP3）ダウンロード
- モバイル対応（スマホから操作可能）
- クリップボードから貼り付けボタン
